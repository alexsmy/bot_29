# Telegram Hub Bot

## Что добавлено

- Telegram-бот на aiogram 3;
- Главное меню «Модульный Хаб»;
- Раздел автоподдержки;
- Раздел файлового хранилища;
- Заготовка шифратора;
- Интеграция радио;
- Интеграция сборщика;
- Модуль часов;
- Единая навигация;
- Модульная архитектура.

## Структура

services/telegram_hub/

- bot_runner.py
- handlers.py
- keyboards.py
- file_service.py
- config.py

## Настройка

Создать переменные окружения:

TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_ADMIN_PIN=1234

## Запуск

Render автоматически поднимет Telegram-бот вместе с FastAPI.

## Проверка

1. Открыть Telegram;
2. Выполнить /start;
3. Проверить разделы;
4. Проверить навигацию;
5. Проверить файловое хранилище;
6. Проверить PIN-доступ;
7. Проверить отсутствие ошибок в Render logs.
